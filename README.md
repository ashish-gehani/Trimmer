# FileIO

#### Installation:
      
     1) Install llvm-3.8.
     2) Install clang-3.8
     3) Update include directory in Makefile
     4) make
 

#### Runing the FileIO Pass
   
     Under ./bin run.sh executes the File IO pass. It takes two arguments 

     a) Input bitcode (.bc)
     b) Output bitcode (.bc)
